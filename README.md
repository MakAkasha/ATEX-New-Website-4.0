# ATEX New Website 4.0

Official website project for **ATEX** — a Saudi technology company focused on delivering smart, scalable IoT and automation solutions for homes, hotels, offices, and enterprise facilities.

## About ATEX

ATEX provides integrated smart solutions that combine:

- Smart devices and field hardware
- Connectivity protocols and secure networking
- Centralized monitoring dashboards
- Automation workflows and operational analytics

The platform is designed to improve comfort, efficiency, and operational visibility across key sectors in Saudi Arabia.

## Technology Stack

- Node.js + Express
- EJS server-rendered views
- Vanilla JavaScript
- CSS (responsive RTL UI)
- SQLite (better-sqlite3)
